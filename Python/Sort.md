## Sort

* sort, sorted 비교

| sort               |       sorted       |
| ------------------ | :----------------: |
| 원본의 순서를 정렬 |  정렬된 값을 반환  |
| 반환 값이 None     | 원본에 영향이 없음 |

* sorted()의 key 인자에 함수 넣기

  * 기준이 하나인 경우

    Sorted(a, key = lambda x : x[0])

  * 기준이 복수인 경우 = 튜플

    Sorted(a, key = lambda x : (x[0], -x[1]))

    #첫 번째 인자로 오름차순 정렬

    #두 번째 인자의 역순으로 오름차순 정렬 == 두 번째 인자로 내림차순 정렬
